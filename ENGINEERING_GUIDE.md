## 前端功能详解

### 标签页 1: 熔池与晶粒
**功能**:
- 主Canvas显示熔池和晶粒生长
- 下方Canvas显示3种视图模式切换:
  - **晶粒尺寸**: 绿色=细小, 红色=粗大
  - **取向分布**: HSL色环表示取向角度
  - **冷却速率**: 蓝色=高冷速, 红色=低冷速
- 晶界显示开关
- 实时监控面板 (7项指标)

---

### 标签页 2: 取向分布图
**功能**:
- **极坐标极图**: 以极坐标形式展示晶粒取向分布
- 圆环分度线 (4圈)
- 8个角度方向参考线 (45°间隔)
- 每个取向区间用不同颜色圆点表示密度
- 自动统计总晶粒数

**颜色映射**:
- 0°: 红色
- 90°: 绿色
- 180°/-180°: 青色
- 270°/-90°: 品红色

---

### 标签页 3: 凝固过程动画
**功能**:
- **自动录制**: 模拟过程中每5步自动保存一帧
- **播放控制**: 播放/暂停/重置
- **速度调节**: 0.5x ~ 3.0x
- **实时显示**: 当前帧数 / 总帧数, 模拟时间
- 独立Canvas渲染动画帧

**使用流程**:
1. 点击"开始模拟"运行模拟
2. 系统自动录制动画帧
3. 模拟完成后切换到"凝固过程动画"标签
4. 点击播放按钮查看凝固过程

---

### 标签页 4: 工艺优化
**功能模块**:

#### 4.1 综合评分卡
- 总分 / 100分
- 等级评定 (A/B/C/D/F)
  - A: ≥90分 (优秀)
  - B: 80-89分 (良好)
  - C: 70-79分 (一般)
  - D: 60-69分 (较差)
  - F: <60分 (不合格)
- 等级颜色动态变化

#### 4.2 力学性能预测 (6项)
| 指标 | 单位 | 说明 |
|------|------|------|
| 屈服强度 | MPa | 基于Hall-Petch公式 |
| 抗拉强度 | MPa | 屈服强度×1.3~1.4 |
| 延伸率 | % | 晶粒尺寸+柱状晶影响 |
| 硬度 | HV | 屈服强度/3.5 |
| 冲击能量 | J | 晶粒尺寸+柱状晶影响 |
| 疲劳极限 | MPa | 屈服强度×0.45 |

#### 4.3 分项评分条 (4项)
| 项目 | 权重 | 目标范围 | 最优值 |
|------|------|----------|--------|
| 屈服强度 | 30% | 350-550 MPa | 450 MPa |
| 延伸率 | 25% | 15-30% | 22% |
| 晶粒尺寸 | 25% | 5-20 μm | 10 μm |
| 柱状晶比例 | 20% | 30-70% | 50% |

- **绿色**: ≥80%权重分
- **黄色**: 60-80%权重分
- **红色**: <60%权重分

#### 4.4 优化建议列表
按优先级排序 (高/中/低/信息):
- 🔴 **高优先级**: 强度/塑性严重不达标
- 🟡 **中优先级**: 柱状晶比例过高
- 🔵 **低优先级**: 性能微调建议
- 🟢 **信息**: 综合评估说明

每条建议包含:
- 问题描述
- 具体参数调整建议 (焊接速度/热输入)

#### 4.5 推荐参数面板
- 推荐的焊接速度、热输入、预热温度
- 预期效果 (晶粒尺寸、强度、延伸率、评分)
- 适用场景说明

## 后端核心算法

### MechanicalPropertyPredictor (力学性能预测器)

#### Hall-Petch 强化机制
```
σ_y = (σ₀ + k_y / √d) × (1 - 0.15 × f_columnar)

其中:
- σ₀ = 150 MPa (晶格摩擦应力)
- k_y = 0.75 MPa·m^0.5 (晶界强化系数)
- d = 晶粒尺寸 (μm)
- f_columnar = 柱状晶比例
```

#### 性能预测公式
| 性能 | 公式 |
|------|------|
| 抗拉强度 | UTS = σ_y × (1.3 ~ 1.4) |
| 延伸率 | δ = 25 + max(0, 10 - 2d) - 8×f_columnar |
| 硬度 | HV = σ_y / 3.5 |
| 冲击能量 | CVN = 50 + max(0, 3d) - 25×f_columnar |
| 疲劳极限 | σ_fatigue = σ_y × 0.45 |

---

### ProcessOptimizer (工艺优化器)

#### 评分算法
总分 = 屈服强度得分(30%) + 延伸率得分(25%) + 晶粒尺寸得分(25%) + 柱状晶比例得分(20%)

#### 单项目得分公式
```
在目标范围内:
  score = weight × (1 - 0.5 × |value - optimal| / (range/2))

低于最小值:
  score = weight × max(0, 1 - 2 × (min - value)/min)

高于最大值:
  score = weight × max(0, 1 - 2 × (value - max)/max)
```

#### 优化建议生成规则
| 条件 | 建议 |
|------|------|
| 屈服强度得分 < 60% | 提高焊接速度30% / 降低热输入15% |
| 延伸率得分 < 60% | 降低焊接速度25% / 提高热输入15% |
| 柱状晶比例得分 < 60% | 提高焊接速度20% |
| 屈服强度得分 > 90% | 可适当降低焊接速度10% |

---

### GrainStatisticsAnalyzer (晶粒统计分析器)

#### 统计指标
- 均值、中位数、标准差
- 最小值、最大值、极差
- 偏度 (分布对称性)
- 10区间直方图

#### 冷却速率-晶粒尺寸相关性
```
皮尔逊相关系数 (Pearson Correlation):

r = Σ[(x-x̄)(y-ȳ)] / √[Σ(x-x̄)²Σ(y-ȳ)²]

解释:
|r| ≥ 0.8: 强相关
0.5 ≤ |r| < 0.8: 中等相关
0.3 ≤ |r| < 0.5: 弱相关
|r| < 0.3: 无相关
```

## 使用指南

### 标准工作流程

1. **设置参数**
   - 调节焊接速度 (1-20 mm/s)
   - 调节热输入 (2-25 kJ/cm)
   - 调节预热温度 (0-500 °C)

2. **运行模拟**
   - 点击"开始模拟"
   - 观察晶粒生长过程
   - 系统自动录制动画帧

3. **查看结果**
   - 切换到"取向分布图"查看晶粒取向
   - 切换到"凝固过程动画"回放过程

4. **分析优化**
   - 点击"分析预测"按钮
   - 查看综合评分和力学性能预测
   - 根据优化建议调整参数
   - 重新模拟验证效果

5. **保存数据**
   - 点击"保存参数"
   - 点击"保存快照"
   - 点击"加载历史"查看记录

## 故障排查

### 常见问题

**Q: 模拟运行卡顿?**
- 降低网格分辨率 (当前200×200)
- 关闭"晶界显示"选项
- 使用Chrome浏览器 (Canvas性能更优)

**Q: 评分总是很低?**
- 模拟需要运行足够时间 (晶粒充分生长)
- 焊接速度建议在5-15 mm/s范围
- 热输入建议在6-12 kJ/cm范围

**Q: API请求失败?**
- 确认服务器运行在端口8088
- 检查浏览器控制台错误信息
- 查看服务器日志输出

**Q: 取向图空白?**
- 需要先运行模拟产生晶粒
- 模拟运行到焊枪经过一半位置以上效果最佳
